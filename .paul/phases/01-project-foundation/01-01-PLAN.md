<objective>
## Goal
Scaffold the Turborepo monorepo with NestJS API app, Next.js web app, and shared packages (database, types, config). All apps install and build cleanly.

## Purpose
Establishes the project structure that all subsequent phases build into. Every module, service, and component lives within this scaffold.

## Output
- Root monorepo with pnpm workspaces + Turborepo
- `apps/api` — NestJS app that starts on port 3001
- `apps/web` — Next.js App Router app that starts on port 3000
- `packages/database` — Prisma package (schema comes in Plan 01-02)
- `packages/types` — Shared Zod schemas package
- `packages/config` — Shared tsconfig bases
</objective>
